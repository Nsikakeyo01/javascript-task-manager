# JavaScript Task Manager

## Project Description
This project is a simple Task Manager web application built using JavaScript, HTML, and CSS. It allows users to add, delete, and mark tasks as completed. The project demonstrates key JavaScript programming concepts including ES6 array methods, recursion, exception handling, and use of external libraries.

---

## Purpose of the Project
The purpose of this project is to demonstrate understanding of core JavaScript programming concepts required in CSE 310. It also helps users manage daily tasks efficiently while applying programming principles learned in the course.

---

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)
- SweetAlert2 (External Library)

---

## Features
- Add new tasks
- Delete tasks
- Mark tasks as completed
- Display task list dynamically
- Input validation with error handling
- Task statistics using ES6 array methods
- Recursive function example for task counting
- Interactive alerts using SweetAlert2

---

## JavaScript Concepts Demonstrated
- Variables and data structures (arrays and objects)
- Functions
- DOM manipulation
- ES6 array methods (map, filter, reduce)
- Recursion
- Exception handling (try/catch and throw)
- External JavaScript library integration
